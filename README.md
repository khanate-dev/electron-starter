# Electron React Starter

Electron desktop application starter

## Building Blocks

- `Electron` with `Electron Forge`
- `React`
- `Typescript`
- `React Router`
- `MUI 5`
- Formatting with `Prettier`
- Linting with `ESLint`
- Spell checking with `cSpell`
- `pnpm` for package management

---

## Scripts

Start: `pnpm start`
